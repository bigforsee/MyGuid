MySQL安装脚本包含功能和限制(本脚本适用于Centos 以及redhat 等能够使用yum 安装软件的操作系统)

DBA 同事提供了一个mysqlinstall.sh 脚本用于安装和优化mysql 服务，相对于之前的mysql.sh 脚本，mysqlinstall.sh 脚本能够一个命令完成数据库安装、数据库优化、数据库密码设置等功能，具体的功能以及安装步骤如下:

功能：

1.能够选择安装MySQL 5.7或者8.0(目前mysql5.7 有官方未修复的漏洞，所以除非客户要求否则后续强烈建议使用mysql8.0)

2.开放当前服务器firewalld 防火墙的mysql 3306端口

3.关闭selinux

4.校验是否已安装mysql rpm包，如果已安装则退出

5.校验指定MySQL安装目录是否为空，不为空则退出，不存在则创建

6.添加优化配置，安装完毕只需要调整innodb_io_capacity和innodb_io_capacity_max参数

7.指定root用户密码

8.显示安装过程

限制：

1.需要网络通畅

2.需要有可用yum源

3.默认采用utf8mb4字符编码

4.因不确定服务器是否只供MySQL使用，innodb_buffer_pool_size只使用了不到50%的内存，对于服务器是专用于此Mysql的可以调整innodb_buffer_pool_size=XG（X为内存总大小的70%-80%）

 

说明: 通过此脚本进行安装mysql 后续不用在进行数据库优化了，此脚本执行完成之后会自动地创建一个已经优化好的my.cnf 文件并替换/etc/my.cnf 文件。 

安装步骤

1.下载MySQL安装脚本

wget  http://dl.qiyuesuo.com/private/mysql/mysqlinstall.sh

2.执行脚本（5.7使用1，8.0使用2）

/bin/bash mysqlinstall.sh

 

3.输入自定义MySQL安装路径，默认为/mysql， 如果指定的目录不存在则自动创建(最好由脚本自动创建或者保证指定的目录为一个空目录)

注意: 这个安装位置目录是指mysql 的数据存储目录，主要存储的是mysql 日志以及数据库数据文件等这个目录最好保证是一个存储空间比较大的目录，防止后续磁盘占满后要迁移。

4.指定root用户密码，至少8位，至少包含大小写字母、数字、特殊字符中的三者（否则之后重启数据库会提示密码错误无法登录） 如果出现此问题请参考知识树修改密码

注意：一定要客户记录好root密码，安装脚本不会保存root密码

 

5.安装成功 ，请根据服务器磁盘类型优化/etc/my.cnf文件末尾的innodb_io_capacity和innodb_io_capacity_max参数

 

# innodb_io_capacity_max一般设置为innodb_io_capacity的2倍，对于SSD硬盘，innodb_io_capacity可以设置8000更高甚至上万的值，对于普通SAS硬盘，可设置200，对于sas raid10可设置2000，对于fusion-io闪存设备可设置几万以上，注意：此参数对于数据库性能影响很大，根据实际磁盘类型进行调整。

 

6.改完参数后重启数据库服务，至此安装结束

systemctl restart mysqld

 

7.登录数据库

 

注: 由于上述mysql 模板中修改了一些配置，故而登录后的mysql 提示符和平常的"mysql >" 不一样，上述脚本执行后，sql 界面的提示符格式如下:

 

 

8. 创建数据库并赋权

 

# 使用 root用户登陆数据库，执行 sql 命令,创建普通用户
# 其中 'qiyuesuo'代表用户名，'%'代表可以使用该用户访问的 ip 不受限制，‘Admin#1234’代表该用户的访问密码
CREATE USER qiyuesuo@'%' IDENTIFIED BY 'Admin#1234';# 其中“Admin#1234”为你自己需要设置的密码，切记要妥善保存

# 创建数据库
# 其中 'qiyuesuodb'为数据库名，'utf8mb4'为默认的字符编码
create schema qiyuesuodb default character set utf8mb4;

# 对用户授权
# 其中 'all'代表所有权限，'qiyuesuodb.*'该数据库中所有表
GRANT all ON qiyuesuodb.* TO 'qiyuesuo'@'%';

#删除drop权限，如果后续代码会升级drop保留

REVOKE drop   ON qiyuesuodb.* from  'qiyuesuo'@'%';

# 刷新权限
flush privileges;
